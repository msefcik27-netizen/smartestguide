🔑 Anthropic API klíč

API klíč není nastaven

Klíč je potřeba pro AI extrakci dat z webů hotelů. Získáš ho zdarma na console.anthropic.com → API Keys → Create Key.

Jak získat API klíč:
1. Jdi na console.anthropic.com
2. Registruj se nebo přihlas
3. Klikni na API Keys v levém menu
4. Klikni Create Key
5. Zkopíruj klíč (začíná sk-ant-) a vlož ho sem

💳 Stripe – platby

Stripe není nastaven

Secret key a payment link pro automatické předplatné hotelů.

Jak nastavit webhook (pro automatickou aktivaci):
1. Stripe Dashboard → Developers → Webhooks → Add endpoint
2. URL: http://tvoje-domena.cz/api/stripe/webhook
3. Events: checkout.session.completed
4. Zkopíruj Webhook signing secret → vlož výše
